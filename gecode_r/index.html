My Gecode/R models

Below are some of my Gecode/R models

Explanation and references are in each specific file. Many of these models are translated from my MiniZinc models.
  • all_different_except_0.rb: Global constraint all_different except 0
  • all_interval.rb: All interval (series) problem (CSPLib problem 7)
  • alpha.rb: Standard alphametic problem (B+A+L+L+E+T = 45 and 19 similiar constraints)
  • coins_grid.rb: Placing coins on a grid (Tony Hurlimann)
  • debruijn_binary.rb: de Bruijn sequences, both "classic" and "arbitrary"
  • devils_word.rb: Devil's Word
  • diet.rb: simple diet problem (operations research)
  • eq20.rb: 20 linear equations (standard benchmark)
  • global_cardinality.rb: Global constraint global cardinality
  • golomb_ruler.rb: Golomb ruler problem (CSPLib number 6)
  • langford.rb: Langford's number problem (CSPLib problem 24)
  • least_diff.rb: Least diff problem, an alphametic puzzle, minimize the difference ABCDE-FGHIJ where A..J is integers 0..9 (all different)
  • map.rb: Simple map coloring problem.
  • minesweeper.rb: Minesweeper problem.
    Problem files for the Minesweeper program.
    Usage: ruby minesweeper.rb problem file

  • nonogram.rb: Nonogram (Paint by Numbers)
    • Problem files for nonogram.rb (most are from Gecode examples)
    • nonogram_simple.txt: Very simple problem
    • nonogram_bear.txt: Bear
    • nonogram_castle.txt: Castle (benchmark, cf http://www.cs.kuleuven.be/~bmd/nonogram.pl, Prolog CLP program)
    • nonogram_crocodile.txt: Crocodile
    • nonogram_difficult.txt: Difficult
    • nonogram_dragonfly.txt: Dragonfly

  • quasigroup_completion.rb: Quasigroup Completion

  • send_more_money_any_base.rb: SEND+MORE=MONEY in any base
  • send_most_money2.rb: SEND+MOST=MONEY, finding all solutions of the max value for MONEY. (Simple extension of the example in the Gecode/R distribution.)
  • seseman.rb: simple recreational mathematics puzzle. See the CGI-program Seseman convent problem
  • set_partition.rb: Set partition problem: sums and sum_squared
  • sliding_sum.rb: Global constrait sliding sum
  • steiner.rb: Steiner triplets.
  • survo_puzzle.rb: Survo puzzle
