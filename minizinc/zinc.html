My Zinc models

Many of these Zinc models are translated from Minizinc model. See My MiniZinc page for more information about MiniZinc, and a lot of MiniZinc models.

Note: the MiniZinc language is a subset of Zinc, so Zinc should be able to run all my MiniZinc models as well. Below are models which use some specific feature of the Zinc language.

I blogged about my first Zinc models in A first look at G12 Zinc: Basic learning models etc
